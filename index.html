🤖 Esecuzione automatica
Il bot gira ogni 15 minuti e pubblica automaticamente dopo ogni partita della Roma, nella finestra -1h / +3h dall'inizio.
🔧 Override manuale
Ignora la finestra temporale e genera le statistiche dell'ultima partita disponibile. Utile per testare o per rigenerare un post già pubblicato.
⏱ Statistiche 1° Tempo
Pubblica le statistiche del primo tempo durante una partita in corso all'intervallo. Funziona solo se la Roma sta giocando e siamo all'intervallo.
ℹ️ Dopo aver cliccato, GitHub Actions si avvia in ~5 secondi. I dati si aggiornano in 30-60 secondi. Clicca "Ricarica dati" per vedere il risultato.

Variabili d'ambiente
Configura su GitHub → Settings → Secrets → Actions:

BSKY_HANDLE — handle Bluesky (es. nome.bsky.social)
BSKY_PASSWORD — App Password Bluesky
IG_USER_ID — ID numerico account Instagram
IG_ACCESS_TOKEN — Token accesso Instagram Graph API

Configura su Vercel → Settings → Environment Variables:

GH_TOKEN — Personal Access Token (scope: repo + workflow)
GH_OWNER — il tuo username GitHub
GH_REPO — nome del repository
Come creare il GitHub Token
1. Vai su github.com → Settings → Developer settings
2. Personal access tokens → Tokens (classic)
3. Generate new token
4. Spunta i permessi: repo e workflow
5. Copia il token e incollalo nella variabile GITHUB_TOKEN su Vercel
